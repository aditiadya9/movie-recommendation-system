# 🎬 Movie Recommendation System

## 📌 Objective
This project aims to build a simple movie recommendation system that suggests movies based on genre similarity and prioritizes them using ratings.

## 🛠️ Tools & Technologies
- Python
- Pandas

## ⚙️ Features
- Takes user input for a movie name
- Recommends movies of the same genre
- Displays movie details (genre and rating)
- Sorts recommendations based on rating
- Handles invalid inputs gracefully

## 📂 Project Structure
Movie-Recommendation/
│── data/ → dataset (movies.csv)  
│── src/ → source code (recommend.py)  
│── outputs/ → (optional future use)  

## ▶️ How to Run
1. Install required library:
   pip install pandas  

2. Run the program:
   python src/recommend.py  

3. Enter a movie name when prompted
